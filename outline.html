SVG Parser:

A user-friendly webapp made with JavaScript, HTML and CSS allows one to view, add, delete and edit attributes of any .svg image. Backend server and API calls are made using Node.js and implemented with C which edit the attributes in real time and update the image(s) in the view panel. A log of these changes are kept in a MySQL database.
